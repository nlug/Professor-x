# Professor-x
API tìm phòng cho các dị nhân giúp đỡ bác giáo sư X
# Hướng dẫn cài đặt
    git clone https://github.com/nlug/Professor-x
    cd Professor-x
    npm install
# Chạy thử
    coffee app.coffee

Truy cập [http://localhost:27000/install](http://localhost:27000/install) để thêm một vài Xmen mẫu
